ZLE WEB – PiApp Studio startovací balíček
=======================================

Struktura složek pro ZIP nahraný do PiApp Studia:

  /index.html
  /css/styles.css
  /js/main.js
  /assets/
    /bg-collage/zle-bg-collage.jpg
    /hero/hero-1.jpg .. hero-6.jpg
    /merch/hoodie-og-zle.jpg
    /merch/tee-ice-cream.jpg
    /merch/cap-stencil.jpg
    /story/story-1.jpg .. story-3.jpg
    /crew/crew-1.jpg .. crew-3.jpg

Postup:
1) Vezmi tento projekt, vytvoř na disku stejné složky a soubory.
2) Do složek /assets/ nahraj reálné fotky ZLE (černobílé / grain, dle našeho stylu).
3) Zabal vše do .zip (bez nadřazené složky, root = index.html atd.).
4) Nahraj ZIP do PiApp Studia jako novou webovou appku.
5) Další produkty/sekce přidáš tak, že rozšíříš sekci #shop a doplníš další karty.

Tento základ je postavený v duchu "ZLE Hybrid Style System v1.0" – černobílé kolážové pozadí, velká typografie, jednoduchý shop, story, crew a kontakt.
